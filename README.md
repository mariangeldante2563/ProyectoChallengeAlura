# ProyectoChallengeAlura

Proyecto Challenge de Alura LATAM

## Descripción
Este es el proyecto del Challenge de Alura, donde se desarrollarán las soluciones propuestas.

## Estructura del Proyecto
```
ProyectoChallengeAlura/
├── src/           # Código fuente principal
├── tests/         # Pruebas unitarias
├── requirements.txt  # Dependencias del proyecto
└── README.md      # Este archivo
```

## Instalación
```bash
# Clonar el repositorio
git clone https://github.com/mariangeldante2563/ProyectoChallengeAlura.git

# Navegar al directorio
cd ProyectoChallengeAlura

# Instalar dependencias
pip install -r requirements.txt
```
